# Simplified Professional Portfolio Website

This repository contains a simplified, professional portfolio website that showcases skills, projects, and blog posts in a clean, easy-to-understand layout.

## Changes Made

- **Simplified Navigation**: Added clear sections for Home, Skills, Projects, Blog, and Contact
- **Skills Section**: Visually appealing grid layout showcasing technical abilities
- **Projects Section**: Clean presentation of projects with clear call-to-action buttons
- **Blog Section**: Dedicated area for blog posts with category tags and dates
- **Improved UX**: Enhanced navigation with active link highlighting and smooth scrolling
- **Responsive Design**: Maintained mobile-friendly layout across all devices

## Features

- Modern, clean design with professional color scheme
- Mobile-responsive layout
- Smooth scrolling navigation
- Interactive elements with hover effects
- Organized content sections for better user experience

## Files

- `index.html`: Main portfolio page with all sections
- `simple-styles.css`: Updated styles including new sections
- `simple-portfolio.html`: Alternative simplified portfolio version
